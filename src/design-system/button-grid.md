Button grids are used for showing groups of 88&times;31 buttons, because that's a common enough situation on this website to warrant a whole component. Retro, baybee.

The component controls the spacing between buttons, hover effects, and link styles whenever links are present.

{{ kimExample("button-grid", "Button grid") }}

## Usage

Use when:

- Displaying one or more 88&times;31 button graphics.

Don't use when:

- Doing literally anything else.
