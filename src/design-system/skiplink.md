A link that only appears when focused using a keyboard or screen reader.

These are navigation aids that allow users to jump directly to specific parts of a page (usually, the main content) without having to go through the header over and over again.

See [the WCAG criterion for bypass blocks](https://www.w3.org/WAI/WCAG22/Understanding/bypass-blocks.html) for more info.
