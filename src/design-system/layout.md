There are a few styles that are used to manage the overall layout of pages. All of them are made available in CSS using the (maybe slightly misnamed) `grid` custom properties.

- `--grid-max-width` controls the maximum width of a page.
- `--grid-gutter` defines the spacing between columns on pages that have multiple columns. It is equivalent to 4 on [the spacing scale]({{ '/design-system/spacing/' | url }}) at the narrowest and 6 at the widest, slightly breaking from the 'one-up' spacing rule.
- `--grid-gutter-half` is the same as the above, but halved, which is useful sometimes.
