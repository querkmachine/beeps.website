---
title: CSS Image Animation Module Level 1
href: https://www.w3.org/TR/css-image-animation-1/
date: 2026-06-10
---

A currently draft CSS specification that would allow for animated GIF, PNG and WebP files to have their playback controlled via CSS properties. Will be really useful for giving users the option to not autoplay them and having user generated content respect `(prefers-reduced-motion)`.
