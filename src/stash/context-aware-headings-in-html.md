An overview of the draft `headingoffset` and `headingreset` HTML attributes, which allow you to change the relative hierarchical position of `<h#>` elements.

I personally find attempts to change this up to often be unintuitive, and the previous major attempt (the Document Outline Algorithm) didn't go anywhere, but we'll see how it goes.
