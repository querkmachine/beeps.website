Scalable Vector Graphics are image files that can be embedded directly within HTML, rather than being included from an external file.

`svg` and its children belong to a separate specification and not strictly HTML elements, but are a reasonably common sight to see on webpages. See [MDN's list of SVG elements](https://developer.mozilla.org/en-US/docs/Web/SVG/Reference/Element) for references on all of those.
