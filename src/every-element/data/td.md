---
tag: td
name: Table Data
category: Tables
spec: https://html.spec.whatwg.org/multipage/tables.html#the-td-element
---

A cell of data contained within a [`table`](#table). Must be the child element of a [`tr`](#tr).

Avoid leaving cells empty, as this can confuse users of some assistive technologies. Add text such as 'Not applicable' if there's no data available.
