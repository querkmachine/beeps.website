---
tag: template
name: Content Template
category: Scripting
spec: https://html.spec.whatwg.org/multipage/scripting.html#the-template-element
---

Container that can hold snippets of HTML code without the code being rendered on the page.

This is useful for JavaScript or when working with [Web Components](https://developer.mozilla.org/en-US/docs/Web/API/Web_components), as it allows the access and use of HTML without needing to manually recreate the HTML in JavaScript.
