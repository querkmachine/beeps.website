---
tag: tr
name: Table Row
category: Tables
spec: https://html.spec.whatwg.org/multipage/tables.html#the-tr-element
---

Container for the table cells elements, [`th`](#th) and [`td`](#td), and delineates where each row of data begins and ends.

`tr` elements must be contained within a [`thead`](#thead), [`tbody`](#tbody) or [`tfoot`](#tfoot) element.
