---
tag: tbody
name: Table Body
category: Tables
spec: https://html.spec.whatwg.org/multipage/tables.html#the-tbody-element
---

Surrounds the main content of a [`table`](#table). This will typically be made up of multiple [`tr`](#tr) elements, each containing one or more [`th`](#th) and [`td`](#td) elements.

If a table lacks a `tbody` element, the browser will implicitly insert one.
