**The Totally Awesome Gay, Transgender, Autistic Guide to Making an Awesome Website** (or <abbr>TAGTAGMAW</abbr>, if you're hip with the lingo) is a free 'course' (it's not really a course) covering the basics of HTML, CSS and JavaScript.

This is aimed at the kinds of critters who want to make a website but find most existing resources either too overwhelming or too focused on using libraries like React, which isn't helpful at all if you're just trying to make a nice, simple website.
